CSS Flexbox
In the world of web development, creating flexible and responsive layouts is crucial to providing a seamless user experience across various devices. One powerful tool that has revolutionized the way we approach layout design is CSS Flexbox. In this guide, we'll explore everything you need to know to master Flexbox and elevate your web design skills.

What is Flexbox?
CSS Flexible Box Layout Module, commonly known as Flexbox, is a layout model that allows you to design complex layouts with ease. Unlike traditional layout methods, Flexbox provides a more efficient way to distribute space among items in a container, even when their size is unknown or dynamic.

Main Concepts of Flexbox

1. Flex Container Properties: Flexbox introduces several properties that control the layout of the flex container, such as `justify-content`, `align-items`, `flex-direction`, and `flex-wrap`. These properties allow you to define how flex items are positioned and aligned within the container.

2. Flex Item Properties: Flex items can also be customized using properties like `flex-grow`, `flex-shrink`, and `flex-basis`. These properties determine how flex items grow or shrink to fill available space and establish their initial size.

3. Flexbox Alignment: Flexbox provides powerful alignment capabilities, allowing you to vertically and horizontally align flex items within the container. This is achieved using properties like `justify-content` and `align-items`, as well as their respective shorthand `align` and `justify` properties.

FlexBox : Defualt [display: 'flex']

To begin using Flexbox, you simply need to apply the `display: flex;` property to a container element. This tells the browser to treat the container's children as flex items and enables Flexbox properties to control their layout.

FlexBox Center

The justify-content: center property centers the flex items along the main axis. This means that the items will be aligned in the middle of the container, with equal space on either side.
